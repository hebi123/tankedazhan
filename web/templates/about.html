坦克大战 - 经典重制版

🎮 游戏介绍

《坦克大战》是一款经典的俯视视角 2D 坦克对战游戏。玩家操控坦克在由砖块、钢板、草地等元素组成的关卡中与多波敌人战斗，保护基地，通过击败敌人获得分数。

本游戏采用现代 Web 技术重制，支持多种敌人类型、完整的碰撞系统、实时 HUD 显示，为玩家带来流畅的游戏体验。

✨ 游戏特色

多种敌人类型
4种不同特性的敌人：普通、快速、重装、特殊坦克，每种都有独特的移动速度和射击模式
完整碰撞系统
精确的子弹与瓦片碰撞检测，支持砖块破坏、钢板阻挡等物理交互
波次敌人系统
敌人按波次出现，每波包含不同类型的敌人，增加游戏挑战性
实时 HUD 显示
显示生命值、分数、关卡进度、敌人数量等关键游戏信息
响应式设计
支持不同分辨率和设备，提供最佳的游戏体验
可定制设置
支持音量调节、按键映射、分辨率设置等个性化选项

🛠️ 技术栈

Python 3.10+ Flask 3.1 JavaScript ES6 HTML5 Canvas CSS3 Web API LocalStorage

后端采用 Python Flask 框架提供 RESTful API，前端使用原生 JavaScript 和 HTML5 Canvas 实现游戏渲染，利用现代 Web 技术确保跨平台兼容性。
